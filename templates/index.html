Systemkonfiguration

Stellen Sie zunächst sicher, dass die Kamera den Einfahrtsbereich der Waschstraße vollständig abdeckt. Justieren Sie ggf. die Position der Kamera im Portalrahmen, bis das Kamerabild optimal ausgerichtet ist.

Klicken Sie auf „Neue Konfiguration erstellen“ um das Kamerabild und die Grafikanzeigen zu konfigurieren. Beim Start der Konfiguration wird das System in den Wartungsmodus versetzt, was bis zu 20 Sekunden in Anspruch nehmen kann. Anschließend werden Sie durch die einzelnen Konfigurationsschritte geführt.

Alternativ kann eine zuvor gespeicherte Konfigurationsdatei geladen und an das System übertragen werden. In diesem Fall startet das System automatisch neu und übernimmt die geladene Konfiguration.
